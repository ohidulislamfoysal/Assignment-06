Digitools – AI-Powered Digital Workflow
Digitools হলো এমন একটা প্ল্যাটফর্ম যেখানে ক্রিয়েটর এবং প্রফেশনালরা তাদের প্রয়োজনীয় সব Premium AI Tools, Design Assets এবং Productivity Software একসাথে পাবেন। আমাদের মূল লক্ষ্য হলো ডিজিটাল কাজগুলোকে আরও দ্রুত এবং স্মার্ট করে তোলা।

***Technology Stack***
-এই প্রজেক্টটি তৈরি করতে আমি যে টুলসগুলো ব্যবহার করেছি:
i) React.js: পুরো ওয়েবসাইটটি ডাইনামিক করার জন্য।
ii) Tailwind CSS: সুন্দর এবং কাস্টম ডিজাইনের জন্য।
iii)DaisyUI: রেডিমেড এবং ক্লিন UI কম্পোনেন্ট ব্যবহারের জন্য।
iv)React Icons: প্রফেশনাল আইকন দেখানোর জন্য।
v)React Toastify: ইউজারকে নোটিফিকেশন বা অ্যালার্ট দেখানোর জন্য।

***Key Features***
1. Smart Product Showcase
-এখানে প্রিমিয়াম সব ডিজিটাল টুলস খুব সুন্দরভাবে সাজানো আছে। ইউজাররা "New" বা "Popular" ট্যাগ দেখে খুব সহজেই তাদের দরকারি টুলটি খুঁজে নিতে পারবেন।
2. Dynamic Shopping Cart
-একদম রিয়েল-টাইম কার্ট সিস্টেম। ইউজাররা যেকোনো প্রোডাক্ট পছন্দ হলে তা কার্টে যোগ করতে পারবেন, মোট খরচ দেখতে পারবেন এবং খুব সহজেই চেকআউট সম্পন্ন করতে পারবেন।
3. Fully Responsive UI
-মোবাইল হোক বা ডেস্কটপ—সব ডিভাইসেই ডিজাইনটি খুব সুন্দর দেখায়। স্পেশালি আমাদের প্রিমিয়াম কালার থিম এবং স্মুথ অ্যানিমেশন ইউজারদের একটা দারুণ এক্সপেরিয়েন্স দেয়।
